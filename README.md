# forefront-lz

A landing zone based on the official [Azure Landing Zone project](https://github.com/Azure/terraform-azurerm-caf-enterprise-scale/wiki/%5BUser-Guide%5D-Provider-Configuration), for quickly deploying an environment in Azure to host workloads that will be accredited for handing UK OFFICIAL SENSITIVE data.

This project is for my own personal readiness and to help demonstrate the art of the possible for Microsoft customers and partners to accelerate their journey to the Cloud. 

Use at your own risk.
